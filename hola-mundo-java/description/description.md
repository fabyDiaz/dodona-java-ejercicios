## Hola Mundo Personalizado

Escribe un programa en Java que muestre exactamente este mensaje:
```
Hola, soy Maria!
```

### Importante
La salida debe coincidir exactamente, sin texto extra ni espacios adicionales.

### Input
No hay.

### Output
```
Hola, soy Maria!
```
```

---

## Pasos para subir a Dodona

1. **Crea un repositorio Git** (en GitHub, GitLab, etc.) con la estructura anterior
2. En Dodona, ve a **Admin → Judges** y verifica que el judge de Java esté disponible
3. Ve a **Admin → Repositories** y agrega tu repositorio
4. Una vez sincronizado, busca el ejercicio y agrégalo a tu serie
```
